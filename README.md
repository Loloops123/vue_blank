# Vue 3 Шаблон

Это шаблон для проекта на Vue 3, использующий Vite, TypeScript, Pinia и другие современные инструменты.

## Особенности

- **Vue 3:** Последняя версия прогрессивного JavaScript-фреймворка.
- **Vite:** Быстрый инструмент сборки, обеспечивающий молниеносную разработку.
- **TypeScript:** Типизированное надмножество JavaScript, которое компилируется в обычный JavaScript.
- **Pinia:** Библиотека для управления состоянием в Vue 3, простая и интуитивно понятная.
- **Vue Router:** Официальный маршрутизатор для Vue.js.
- **Tailwind CSS:** Утилитарный CSS-фреймворк для быстрой разработки пользовательского интерфейса.
- **Storybook:** Инструмент для разработки и тестирования UI-компонентов в изоляции.
- **Vitest:** Молниеносно быстрый фреймворк для модульного тестирования, работающий на Vite.
- **ESLint:** Плагин и настраиваемый инструмент для выявления и сообщения о шаблонах в JavaScript.
- **Prettier:** Строгий форматер кода.
- **Husky & lint-staged:** Инструменты для запуска скриптов перед коммитами.

## Начало работы

### Требования

- [Node.js](https://nodejs.org/en/) (версия указана в `package.json`)
- [pnpm](https://pnpm.io/)

### Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Loloops123/vue_blank.git
    ```
2.  Установите зависимости:
    ```bash
    pnpm install
    ```

### Разработка

Для запуска сервера разработки выполните следующую команду:

```bash
pnpm dev
```

Это запустит сервер разработки Vite и откроет приложение в вашем браузере по адресу `http://localhost:5173`.

### Сборка

Для сборки приложения для продакшена выполните следующую команду:

```bash
pnpm build:ci
```

Это создаст директорию `dist` с оптимизированной производственной сборкой приложения.

## Доступные скрипты

- `pnpm dev`: Запускает сервер разработки.
- `pnpm storybook`: Запускает сервер разработки Storybook.
- `pnpm test:unit`: Запускает модульные тесты.
- `pnpm test:storybook`: Запускает тесты Storybook.
- `pnpm lint:all`: Проверяет и форматирует код.
- `pnpm build:dev`: Собирает приложение для разработки.
- `pnpm build:ci`: Собирает приложение для продакшена.
- `pnpm storybook:build`: Собирает Storybook для развертывания.
- `pnpm prettier:format`: Форматирует код с помощью Prettier.
- `pnpm knip`: Находит неиспользуемые файлы, зависимости и экспорты в ваших проектах JavaScript и TypeScript.
- `pnpm prepare`: Настраивает Husky для pre-commit хуков.
